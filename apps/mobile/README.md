# mobile

Expo Router mobile app scaffold for dumcont.

## Environment

Copy `.env.example` to `.env` and configure:

- `EXPO_PUBLIC_API_BASE_URL`
- `EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY`

## Scripts

- `npm install`
- `npm run dev`
- `npm run typecheck`
- from repo root: `npm --prefix apps/mobile run dev`

## Current scope

- Expo Router app shell
- Clerk provider wiring
- Query client provider
- App lock provider scaffold
- Auth, locked, contacts, archived, and settings route groups
